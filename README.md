# Simple library 라이브러리 만들기

- windows API
  - Sp01_api
- direct3d9 device
  - Sp02_device
- d3dx9 sprite
  - Sp02_sprite
- texture and matrix
  - Sp03_texture
  - Sp03_texture_ani sprite
  - Sp03_Texture_matrix
  - Sp03_texture_numerous
- 샘플 게임
  - Sp05_puzzle
  - Sp05_puzzle_complete
